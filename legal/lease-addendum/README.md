# Lease Addendum — International Student Verification

**Status:** Not yet drafted — placeholder

---

## What belongs here

A lease addendum template that landlords can attach to standard Utah residential
lease agreements, replacing the SSN/credit-check requirement with BridgeStay's
passport and I-20 verification process.

## Purpose

The core value proposition of BridgeStay is enabling international students to
secure housing without a Social Security Number or U.S. credit history.
This addendum is the legal instrument that makes that possible.

## Key clauses to include

- Statement that standard SSN/credit check requirement is waived for verified
  BridgeStay tenants
- Definition of "BridgeStay Verified Tenant" (passport + I-20 + admin review)
- Landlord acknowledgment of BridgeStay verification process
- Alternative security deposit terms (if applicable)
- Governing law (Utah Residential Landlord and Tenant Act)
- Signatures from landlord and tenant

## Suggested filename

`BridgeStay_Lease_Addendum_Template_v1.0.docx` (editable template)
`BridgeStay_Lease_Addendum_Template_v1.0.pdf` (reference copy)

## Notes

- Must comply with the Utah Fit Premises Act and Utah Code Ann. § 57-22
- Consult a Utah real estate attorney before distributing to landlords
- This is a template — each signed copy should be stored privately,
  not committed to version control
